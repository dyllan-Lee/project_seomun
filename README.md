# project_seomun


html5 ,css3 에 대해서 배우고 가장 먼저 제작해본 웹제작페이지 입니다. 

기존의 서문시장 야시장 페이지를 분석하고 제 스타일로 해석하여 요소들을 재 배치하여 만든 페이지입니다.
